# 已确认的生产日志路径

## 项目根目录

- 当前 `app-backend` 机器已确认项目根目录为 `/opt/magicain`

## app-backend

- 机器职责：`Redis + Cloud`
- Compose 文件：`docker-compose.app-backend.yml`
- 宿主机日志目录：`/opt/magicain/logs/cloud`
- 当前已确认主日志文件：`/opt/magicain/logs/cloud/pangju-server.log`
- 容器内挂载目录：`/root/logs`

## app-frontend

- 机器职责：`Nginx + 3 个前端 UI`
- Compose 文件：`docker-compose.app-frontend.yml`
- 宿主机日志目录：`/opt/magicain/logs/nginx`
- 典型日志文件：
  - `/opt/magicain/logs/nginx/access.log`
  - `/opt/magicain/logs/nginx/error.log`
- 容器内挂载目录：`/var/log/nginx`

## 使用约定

- 在 `app-backend` 机器上优先查 `logs/cloud`
- 在 `app-frontend` 机器上优先查 `logs/nginx`
- 不要在 `app-backend` 机器上找 Nginx 日志
